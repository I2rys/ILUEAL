## Installation
Github:

    git clone https://github.com/I2rys/ILUEAL
    
NPM Packages:
```
npm i localtunnel && npm i request-ip && npm i express
```

## Usage
Default start:
```
node index.js
```

NPM start:
```
npm start
```

## Dev Usage
NPM start dev:
```
npm run start:dev
```
